Project README

Tater Tunnel is a privacy-first secure networking layer for Tater. The user brings a VPS; Tater Tunnel pairs a Home Agent to that VPS over a simple outbound relay path, while remote phones and laptops can use WireGuard VPN mode against the VPS.

VPS Install

Tater Tunnel has two VPS setup paths.

Trust Boundaries

The VPS relays traffic. The Home Agent owns trust.

MVP Spec

Secure remote access to Tater without port forwarding, public dashboards, cloud tunnel accounts, or advanced networking setup.

# Tater Tunnel Image Prompts

These are the prompts used for the first website image set.

## Hero

Use case: ads-marketing
Asset type: website hero image for Tater Tunnel
Primary request: Create a polished hero image of the Tater mascot trying to plug two ethernet cables together, matching the mascot style from the visible Tater logo/reference: friendly potato head, black visor sunglasses, round headphones, white rounded robot suit, glossy black gloves, orange glow accents. The mascot is centered and smiling, holding one ethernet cable in each hand with the RJ45 plugs almost touching and a small warm spark/glow between them.
Scene/backdrop: dark clean cyber/network backdrop with subtle grid lines, orange and cool blue connection traces, no text.
Style/medium: high-quality 3D mascot render, brand asset, crisp and friendly.
Composition/framing: wide landscape hero composition, subject large and clear, some negative space on left and right for website layout, cables stretching outward.
Lighting/mood: warm orange rim light, soft studio highlights, secure and helpful mood.
Color palette: black, white, potato gold, tater orange glow, restrained cool blue network accents.
Constraints: no readable text, no logo text, no watermark, no extra characters, no scary or aggressive tone. The ethernet plugs must be visibly RJ45 network plugs.

## VPS Setup

Use case: ads-marketing
Asset type: website support image for setup section
Primary request: Create a polished 3D image of the same Tater mascot style from the visible reference, standing beside a small VPS server tower and a glowing shield/lock symbol, holding a simple setup clipboard. The mascot should feel helpful and friendly, like it is guiding a user through a secure VPS setup.
Scene/backdrop: dark minimal network workspace with subtle orange and blue connection lines, no text.
Style/medium: high-quality 3D mascot render, brand website illustration.
Composition/framing: landscape card-friendly composition, mascot on one side, VPS/shield visual on the other, clean empty breathing room.
Lighting/mood: warm tater orange glow, calm secure mood.
Color palette: potato gold, white robot suit, black visor/gloves, tater orange, restrained cool blue.
Constraints: no readable text, no watermark, no extra characters, avoid clutter.

## Mobile QR

Use case: ads-marketing
Asset type: website support image for mobile VPN section
Primary request: Create a polished 3D image of the same Tater mascot style from the visible reference, holding a smartphone that shows a simple QR-like square pattern on screen while a glowing WireGuard-style VPN connection line arcs from the phone to a small VPS node. Make it clear this is a scan-and-connect mobile VPN setup, but do not include the WireGuard logo or any real brand marks.
Scene/backdrop: dark clean network backdrop with orange and blue light traces, no readable text.
Style/medium: high-quality 3D mascot render, friendly product website illustration.
Composition/framing: landscape card-friendly composition with the phone and QR visible, mascot smiling, network path visible.
Lighting/mood: warm secure onboarding mood.
Color palette: potato gold, white robot suit, black visor/gloves, tater orange, restrained cool blue.
Constraints: no readable text, no watermark, no extra characters, no real app logos or brand marks.
